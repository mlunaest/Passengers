All the code is in only one file, a7.py and uses the pymssql library. 
It requires python 2.7 to run and is a simple command line application.

The program is quite straight forward, and all menus require input of numbers, but they are clearly labeled as such.
to navigate. Input into functions is assumed to be in correct syntax: for example, the depart date must be input in the format '2016-11-28'. 
Upon completion of a function, the program will return to the main menu until the user chooses to terminate the program.
